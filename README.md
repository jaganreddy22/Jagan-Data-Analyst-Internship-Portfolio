## Internship Tasks

### Task 1 – Data Immersion & Wrangling

Deliverables:

• Clean Dataset

### Task 2 – Exploratory Data Analysis
Analyzed patterns and trends in sales and profitability using Python and SQL.

Key Insights:
• Regional sales trends
• Category performance
• Profit distribution

---

### Task 3 – Deep Dive Analysis & Dashboard
Defined key KPIs and built an interactive dashboard for business insights.

KPIs:
• Total Revenue
• Total Profit
• Profit Margin
• Category Performance

---

### Task 4 – Data Storytelling & Statistical Validation

Business Hypothesis:
High discount levels significantly reduce profitability.

Statistical Test:
Independent Two-Sample T-Test

Result:
P-value = 7.71 × 10⁻¹⁰⁶

Conclusion:
High discount strategies significantly reduce profit margins.
